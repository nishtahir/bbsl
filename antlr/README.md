# ANTLR 4 CPP Runtime

> This is a copy of the source for the convenience of building the project. 
> See the parent repository [here](https://github.com/antlr/antlr4/tree/master/runtime/Cpp) 

This folder contains the C++ runtime support for ANTLR.  See [the canonical antlr4 repository](https://github.com/antlr/antlr4) for in depth detail about how to use ANTLR 4.
